QQ空间备份神器，一键备份你的青春

发表于 2022-01-27 更新于 2022-02-19 分类于 软件工具 ， 备份工具 ， 浏览器扩展 阅读次数： 本文字数： 14k 阅读时长 ≈ 13 分钟
QQ空间导出助手官方教程，一键备份QQ空间的说说、日志、私密日记、相册、视频、留言板、QQ好友、收藏夹、分享、访客

QQ空间导出助手，一键备份QQ空间的说说、日志、私密日记、相册、视频、留言板、QQ好友、收藏夹、分享、访客到文件

落叶随风，青春，稍纵即逝，QQ空间，一个承载了很多人的青春的地方。

或许，是遗憾，毕竟，谁的青春没留遗憾呢，《曾经沧海无限感慨，唯愿往事随风》。

或许，是害怕，曾经的青春变得不可控，毕竟，新浪博客相册、网易相册、腾讯微博等相继停运，无不意味着，互联网产品都有着自己的生命周期。

于是，萌生了备份QQ空间的念头，也在互联网上找到一些工具与脚本，要么操作复杂、要么备份类型单一，于是乎，简单易用全类型备份的QQ空间导出助手诞生了。

功能清单

  1. 支持备份QQ空间文字说说、图文说说、语音说说、长说说，以及评论、点赞、最近访问
  2. 支持备份QQ空间文字日志、图文日志、模板日志，以及评论、点赞、最近访问
  3. 支持备份QQ空间文字私密日记、图文私密日记，以及评论、点赞、最近访问
  4. 支持备份QQ空间相册/相片，以及相册/相片的评论、相册/相片的点赞、相册的最近访问
  5. 支持备份QQ空间视频，以及评论、点赞、最近访问
  6. 支持备份QQ空间留言板寄语与留言、以及留言回复
  7. 支持备份QQ好友、含好友成立时间、单向好友检测、空间访问权限检测、特别关心的好友等
  8. 支持备份QQ空间分享内容，以及评论、点赞、最近访问
  9. 支持备份QQ空间收藏夹内容
  10. 支持备份QQ空间访客

相关术语

阅前重点 ，阅前重点 ，阅前重点
以下为助手的相关术语，先进行阅读有个总体的概念，后续正式开始备份的前后，可再回来阅读

备份压缩包

助手采集数据完成后，点击下方打包下载后得到的压缩文件，也就是ZIP文件，被称为备份压缩包，文件以QQ空间备份_QQ号命名

文案内容

指QQ空间的纯文字内容，比如说说文字/评论、日志内容/评论、留言的文字、相册名称、相册描述、相片名称、相片描述、点赞列表/人、最近访客等文字内容

文案内容文件

用来保存文案内容以及展示文案内容的实体文件，比如HTML、MD、JSON文件
实体文件文件是根据备份类型生成的文件，存放在备份压缩包中

文案内容下载文件夹

也就是浏览器设置中的下载文件夹，打包下载后，备份压缩包后将自动存放到该文件夹下

Edge如何设置下载文件夹？
Chrome如何设置下载文件夹？
360极速浏览器如何设置下载文件夹？

文案内容备份文件夹

就是存放了所有文案内容文件的文件夹，即解压备份压缩包后得到的文件夹
该文件夹一般存放在文案内容下载文件夹中，除非你特意解压到了其它地方

多媒体文件

实际的文件，比如说说的配图文件、日志的配图文件、留言配图文件等等，语音说说的MP3文件，相册的相片JPEG/JPG/PNG/GIF文件，视频的MP4文件等

多媒体文件下载文件夹

就是助手配置的文件下载工具所设置的下载文件夹
如果你下载工具为浏览器，该文件夹与文案内容下载文件夹为同一个文件夹
如果你下载工具为是迅雷，请查看迅雷设置的下载文件夹

多媒体文件备份文件夹

也就是存放了多媒体文件的文件夹，在多媒体文件下载文件夹中，比如：QQ空间备份_QQ号

完整备份文件夹

同时存放了文案内容与多媒体文件的文件夹，即已经进行备份文件夹合并后的文件夹

  7. 安装完成

使用说明

使用步骤

  1. 先进行助手的安装，更多详情请移步在线安装或离线安装
  2. 助手安装完成后，可以先根据个人情况进行助手的配置，更多详情请移步配置说明
  3. 一起准备就绪，接下来，请先登录QQ空间，登录QQ空间，登录QQ空间，重要的事情说三遍。
  4. 然后，进入需要备份的QQ空间，如果只是备份自己的QQ空间，该步骤可跳过
  5. 备份前，建议刷新需要备份QQ空间页面使助手配置生效，如果是保存配置后再打开的QQ空间页面，该步骤可跳过
  6. 在需要备份的QQ空间页面，插件栏点击插件图标进行备份，如果点击图标无反映请移步这里
  7. 选择要备份的QQ空间内容，建议全部备份，点击开始备份
  8. 助手将开始收集QQ空间的数据，请耐心等待数据收集完成
  9. QQ空间数据采集完成后，助手自行添加多媒体文件的下载任务到文件下载工具
  10. 如若不涉及文案内容的下载，助手的备份任务到此已完成，剩下的请耐心等待文件下载工具把多媒体文件的下载任务完成，然后查看备份内容即可，更多内容移步打包下载说明
  11. 如若涉及文案内容的下载，文件下载工具的下载任务添加完成后，备份界面下方将出现打包下载按钮，更多内容移步打包下载说明
  12. 点击打包下载按钮时，助手自动打包文案内容到备份压缩包并下载
  13. 备份压缩包下载完成后，点击查看备份按钮查看备份压缩包，点击查看备份按钮无法查看是，可自行打开文案内容下载文件夹查看
  14. 再进行文案内容备份文件夹与多媒体文件备份文件夹的合并，详情移步备份文件夹合并
  15. 合并完成后，根据备份类型的不同，自行查看备份内容
  16. 完整备份文件夹结构，如果结构不对，表示合并错误
  17. 最后，你可以把完整备份文件夹复制或移动到其它地方进行再次备份、比如说U盘、网盘、或者电脑的其它位置、甚至部署到服务器，比如备份预览就是基于备份内容部署到服务器的

打包下载说明

如果只备份相册/视频模块，且备份类型为文件/MP4文件时，可能不涉及文案内容下载

一般有以下两个场景：

  1. 文件下载工具不是迅雷下载链接时，不涉及文案内容的备份，无需处理打包下载与备份文件夹合并
  2. 文件下载工具是迅雷下载链接时，不涉及文案内容的备份，需进行打包下载，因迅雷下载链接在备份压缩包，且需进行备份文件夹合并

PS：含其它模块的备份时，均涉及文案内容的下在，也就是需要点击打包下载，比如全部备份，比如说只备份说说、比如同时备份说说、相册(备份类型为文件)

备份文件夹合并

  1. 合并前，请先回顾助手的相关术语
  2. 等待文件下载工具中的多媒体文件的下载任务完成
    1. 浏览器：自行查看浏览器下载内容
    2. 迅雷：自行查看迅雷下载任务情况
    3. Aria2：请自行查看Aria2控制台或使用其他可视化工具，比如Aria2 for Edge或Aria2 for Chrome
    4. Motrix：自行查看Motrix下载任务情况
  3. 复制文案内容备份文件夹，也就是复制QQ空间备份_QQ号的文件夹
  4. 打开多媒体文件下载文件夹，右键粘贴，如若提示文件夹或文件存在，直接覆盖即可
  5. 合并完成，合并后文件夹结构如下
  6. 查看备份内容
  7. 最后，你可以把完整备份文件夹复制或移动到其它地方进行容灾备份、比如说U盘、网盘、或者电脑的其它位置

查看备份内容

根据备份类型的不同，选择不同的查看方式，更多备份类型请查看备份类型
HTML

  1. 浏览器打开相应的HTML文件即可
  2. 建议直接打开QQ空间备份_QQ号文件夹下的index文件

MarkDown

  1. 使用 Typora或 Atom 等Markdown编辑器查看
  2. 建议使用Markdown打开QQ空间备份_QQ号文件夹
  3. 参考：什么是Markdown编辑器

增量备份

  1. 增量备份配置为全量，先进行第一次的备份，参考使用说明
  2. 第一次备份完成后，得到完整备份文件夹，该文件夹是由备份文件夹合并后得到的文件夹，这里简称为文件夹A
  3. 第一次备份完成后，一般有如下场景：
    • 场景①：你把这个文件夹A移动到别的文件夹，比如说U盘的U:\备份，最终形成U:\备份\QQ空间备份_QQ号
    • 场景②：你没有挪动文件夹A，该文件夹仍保留在多媒体文件下载文件夹中
  4. 增量备份配置为上次备份，进行第二次备份，参考使用说明
  5. 第二次备份完成后，将得到完整备份文件夹，该文件夹是由备份文件夹合并后得到的文件夹，这里简称为文件夹B
    • 如果第③步，你采取的是场景①，那直接将文件夹B移动到U:\备份进行覆盖即可，即将文件夹B覆盖文件夹A
    • 如果第③步，你采取的是场景②，你会发现文件夹A和文件夹B其实是同一个文件夹，即已合并完成
  6. 合并完成，最终得到第一次全量备份与第二次增量备份后的完整备份文件夹
  7. 常见的增量备份的场景：
    1. 场景①：第一次所有模块的全量备份后，仍继续使用QQ空间，第二次备份前，把所有模块的增量备份改为上次备份，然后备份即可
    2. 场景②：第一次只备份了说说，其它模块均未备份，第二次想备份日志时，把说说的增量备份改为上次备份，然后备份时，同时勾选说说、日志即可
    3. 场景③：第一次备份时，只勾选了相册模块的A相册，后续想要备份相册模块的B相册，备份前，把相册模块的增量备份改为上次备份，然后备份时，勾选相册模块，然后勾选B相册即可，如果A相册需要增量，把A相册同时勾选，A相册不勾选也会默认保留，但是不会增量备份A相册新增的相片
  8. 注意事项：目前增量备份使用时，不能清除浏览器数据或缓存，不然增量备份或将失效，没实际验证过，或许或会优化
  9. 更多注意事项，请查看配置说明-增量备份

配置说明

如何配置

  1. 浏览器插件栏右键助手图标，选择选项菜单后，将进入助手配置页
  2. 助手配置页顶部分为设置、工具以及关于
    1. 设置：助手配置页的核心页面，在这里进行助手的配置
    2. 工具：一个额外添加本地相册到备份内容中的工具，可忽略
    3. 关于：助手的一些简介与说明
  3. 注意事项：助手更新配置后，需要点击下方配置页的保存按钮
  4. 注意事项：保存后，需要刷新QQ空间页面才能使最新的配置在备份页面生效。

通用配置

备份类型

助手选项->设置->模块->公共选项->备份类型

将QQ空间的文案内容备份到什么类型的文案内容文件，每个模块可单独配置，建议每个模块保持一样的备份类型，比如都为HTML

HTML
浏览器专用文件，直接使用浏览器打开查看即可

MarkDown

  1. 一种轻量级标记语言

账号与隐私

是否会导致QQ空间账号被冻结

  • 一般情况下不会，但如果助手采集速度过快时，偶尔会造成QQ空间短暂的冻结，如造成相册访问空白等。
  • 过段时间会自动解冻，如若介意，请勿使用，否则，后果自负。
  • 如遇冻结，建议上调相应模块的数据采集时间

是否会导致QQ空间账号被封号

  • 理论上不会，但不承诺一定不会封号
  • 至少目前为止，本人没有遭遇封号
  • 曾有用户反馈遭遇封号，如若介意，请勿使用，否则，后果自负。

是否会导致个人隐私的泄露

  • 助手承诺不泄露个人隐私，第三方修改版除外
  • 如因个人原因导致备份文件丢失与泄露，本助手概不负责。
  • 更多详情请访问隐私政策

是否可以导出别人空间的数据

  1. 可以，但仅支持导出有权限访问或公开的数据，即没权限看的就没法备份
  2. 建议先征求别人同意后备份，避免造成误会或造成别人信息泄露
  3. 备份他人空间内容，将会在他人QQ空间访客中留言访问记录，当然，如果你是黄钻当我没说。
  4. 如果造成别人信息泄露，本助手概不负责，更多详情参考隐私政策

是否可以导出别人的加密QQ空间

  1. 有访问权限的，可以，比如别人空间虽然加密了，但是对你开放，可备份
  2. 无访问权限的，不可以，如有大神可以破解，请联系本人！^_^

是否可以导出别人的加密相册

  1. 有访问权限的，可以，比如、别人的相册虽然加密了，但是对你开放，可备份
  2. 有访问权限的，可以，比如、别人的相册虽然有加密问题，但是你回答正确了，可备份
  3. 无访问权限的，不可以，如有大神可以破解，请联系本人！^_^

无法进入QQ空间能否备份QQ空间内容

  1. 不能，助手是模拟用户访问QQ空间进行备份，无法进入时无法备份。
  2. 比如QQ被封号、没有权限访问他人空间等等

是否可以备份私密日记

  1. 可以，但需要先关闭私密日记的【独立密码认证】，可在备份完成后重新打开
  2. 关闭独立密码认证后，刷新重新备份

点击助手图标无反应

  1. 请先登录并进入QQ空间，登录并进入QQ空间，登录并进入QQ空间，重要的事情说三遍，然后再QQ空间的页面再点击图标
  2. 进入了QQ空间页面，点击仍没反映，有可能是因为助手使用了第三方库，第三方服务时常不稳定，可稍后重试，后续版本将切换为离线
  3. 点击有反映，但是出来的界面跟截图不一致，有可能是因为助手使用了第三方库，第三方服务时常不稳定，可稍后重试，后续版本将切换为离线

助手采集过程中，总是有部分数据被跳过，是否正常

  1. 正常，比如说，获取说说全文，如果不是长说说就无需获取全文，就跳过了。

  • 没有正常下载到图片，如QQ空间的图片已不存在或无法正常下载

如何在手机上查看备份内容

  • 安装相应的查看工具即可
  • HTML文件，使用手机自带浏览器或第三方浏览器查看即可，目前手机端适配不好，可能无法正常查看内容，后续会优化
  • MD文件，安装第三方支持MarkDown文件查看的工具即可查看，详情请访问：手机MD编辑器

备份的内容是否支持离线查看，就是没有网络是查看

  1. 备份类型为HTML时，不支持，因为有很多第三方插件是采用在线访问的方式
  2. 备份类型为MarkDown时，支持，但部分表情无法正常查看，因为表情未采用离线方式

查看QQ空间备份数据时，发现数据缺少

  • QQ空间是否能正常查看，如若不能，则正常
  • 如若QQ空间可正常查看，有可能就是助手采集数据频率过快导致的
  • 建议上调相应备份模块的【查询间隔】后刷新QQ空间页面重新备份
  • 更多详情请访问：查询间隔

相冊下载的相片是否为原图

  1. 相册下载时，可指定【相片质量】，默认为原图
  2. 如果QQ空间不存在原图，则依次递减

相冊下载的相片Exif信息丢失

QQ空间本身是否存在原图，原图是否存在Exif信息

相冊下载的相片创建时间为什么不是原图的创建时间

  1. 下载时，无法指定相片的创建时间，默认相片的创建时间为下载文件的时间
  2. 如有Exif信息，则原图的创建时间在Exif信息中

说说下载的配图不是原图

  • 说说发表时，直接上传的配图不存在原图。
  • 如果说说配图来自有原图的QQ空间相册，似乎会有原图，未验证。

日志下载的配图不是原图

  • 日志发表时，直接上传的配图不存在原图。
  • 如果日志配图来自有原图的QQ空间相册，似乎会有原图，未验证。

日志下载的视频不是原画**

似乎是的，因为QQ空间页面上，不存在原画视频下载

助手配置页面错乱，无法正常配置

助手使用了在线访问的第三方库，部分时间或会网络不稳定，可稍后再试

下载工具

浏览器下载

文件下载工具为【浏览器】下载时，不停弹窗提示保存文件，需要一个个的点击保存
请确保已关闭浏览器设置中的下载前询问每个文件的保存位置

  • Edge：设置->下载->每次下载都询问我该做些什么，参考：Edge如何设置下载文件夹
  • Chrome：设置->高级->下载内容->下载前询问每个文件的保存位置，参考：Chrome如何设置下载文件夹
  • 360极速浏览器：选项->下载设置->下载前询问每个文件的保存位置，参考：360极速浏览器如何设置下载文件夹
  • 其它浏览器：请自行摸索

更多详情，可参看文件下载工具-浏览器

文件下载工具为【浏览器】下载时，部分下载文件显示已禁止，且文件名后缀为txt/html
浏览器与QQ空间跨域限制，主要出现在视频文件下载时，无法正常下载，助手暂时无法规避
建议更换文件下载工具为迅雷或Aria2
更多详情，参考文件下载工具-浏览器

迅雷下载

文件下载工具为【迅雷】下载时，无法唤起迅雷下载

  1. 确保已安装官方正版迅雷，版本10以上
  2. 不保证第三方美化、破解、绿色版本可用
  3. 并确保管家类软件没有禁用迅雷的基础服务
  4. 如果还是不行，可将工具更换为【迅雷下载链接】，然后备份完成后，人工打开迅雷添加下载任务
  5. 更多详情，可参看文件下载工具-迅雷

文件下载工具为【迅雷下载链接】下载时，如何下载文件

  1. QQ空间数据采集完成后，点击打包下载
  2. 解压Zip包，解压后根目录有个迅雷下载链接.txt的文件，打开全选复制文件中的内容，然后打开迅雷
  3. 如果迅雷开启了剪切板监听，将会自动添加下载任务
  4. 如果未开启剪切板监听，则自行添加下载任务即可，把复制的内容粘贴到下载地址即可
  5. 更多详情，可参看文件下载工具-迅雷下载链接

Aria2下载

文件下载工具为【Aria2/Motrix】下载时，无法正常添加下载任务

  1. 确保Aria2已安装，版本1.35.0+
  2. 确保Aria2已启动，并开启HTTP方式的RPC服务。
  3. 更多详情，可参看文件下载工具-Aria2

更多工具

助手是否可以支持其他下载工具，如IDM

  1. 暂不支持其他下载工具，因为其他下载工具添加任务时，不支持指定目录的方式添加，而助手的备份内容的设计是含目录结构的
  2. 如果有其他支持指定保存文件夹方式（含目录结构）添加任务的下载工具，可联系我

更多问题

是否支持一键删除QQ空间内容

  1. 不支持，也没有支持的打算，本助手只是备份数据
  2. 已有第三方软件、扩展、脚本支持，可自行搜索

其它未知的、乱七八糟的备份中遇到的问题
可自行先确认下QQ空间的情况，比如本人就常遇到一些乱七八糟的问题

  1. 语音说说的语音忽然备份不了，进入到QQ空间试了试，发现QQ空间的语音也播放不了。
  2. 收藏夹忽然备份不了，然后我一进入QQ空间的收藏夹，一直提示要我登录
  3. 收藏夹中视频忽然备份不了了，一看QQ空间的收藏夹播放视频，发现也播放不了
  4. 说说的配图没法下载，进去QQ空间一看，发现该条说说的配图已经丢失了。
  5. 诸如此类的，有很多，此时暂时想不起来了。

如果QQ空间本身没有问题的，再进行问题反馈
